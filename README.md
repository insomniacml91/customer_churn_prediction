Customer Churn Prediction
ML-пайплайн для предсказания оттока клиентов телеком-оператора.

Бизнес-проблема
Удержание клиента дешевле, чем привлечение нового. Модель предсказывает, уйдет ли клиент в следующем месяце, чтобы маркетологи могли предложить ему скидку.

Технический стек
    Язык: Python
    Обработка данных: Pandas, NumPy
    Моделирование: Scikit-Learn (Random Forest)
    Контроль версий: Git, GitHub

Этапы работы
    Очистка данных: Удаление мультиколлинеарных признаков (TotalCharges), обработка категориальных переменных (One-Hot Encoding, Label Encoding).
    Борьба с дисбалансом: Использование class_weight='balanced' для учета дисбаланса классов.
    Оптимизация: Ограничение глубины дерева (max_depth=5) для предотвращения переобучения.
    Ансамблирование: Применение Random Forest (100 деревьев) для повышения стабильности.

Как запустить
    Клонировать репозиторий.
    Положить файл с данными churns.xls в папку проекта.
    Запустить скрипт: `python train.py
